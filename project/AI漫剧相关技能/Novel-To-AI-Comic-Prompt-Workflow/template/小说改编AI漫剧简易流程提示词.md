## 一、小说内容改写

### 提示词 ① — 小说推文优化

```
你是一个专业的小说推文作者，请你优化这段小说，用作小说推文，
要有爆款开头，结尾要留钩子，不利于观众停留的剧情可以酌情删减掉。
```

### 提示词 ② — 视频推文优化

```
你是一个专业的视频推文博主，请你优化这段文案，用作视频推文，
要有爆款开头，结尾要留钩子，不适合做成视频的内容适当进行改编。
```

## 二、分镜脚本

### 提示词

```
你是一个编剧兼导演，擅长把小说文案改编成适用于 seedance 2.0 视频模型
直接生成视频的分镜脚本。

要求：
- 10 秒为一段
- 最少 3 个分镜
- 每一段固定统一的风格和统一的色调

分镜参考模板：
景别，运镜，主角，动作，台词，音效，光影，场景

风格要求：2D 动漫视频
```

## 三、角色图片生成

### 提示词

```
分析小说里所有的角色，参考以下模板，给出相对应的生图提示词，
要求风格统一。我需要做成高质量 3D 精品漫剧。

---
[人物基础特征]
性别：
年龄：
体型与肤色：
风格：(例如：3D/2D，水墨风、赛博朋克、现代极简、暗黑奇幻)
色彩基调：

[头部细节]
头发：露出颈部和肩膀
脸部：
眉毛：
鼻子：
眼神：

[身体与穿搭]
手部：
腿部：
衣服：要求完全对称，剪裁清晰
裤子：
鞋子：
配饰：无大件遮挡物，小巧贴合

[技术与环境限制]
背景：纯白纯色背景，无纹理、无渐变、无杂物、无任何装饰元素
姿势：绝对标准的 A-Pose，双臂自然下垂，呈 A 字形
比例：哥特式比例，高级时装插画视觉
视角：正前视，平视中心镜头，正交视角，零透视畸变，
      无广角，无仰俯角度，无镜头变形
光照：全局均衡柔和漫射光
构图：完整全身立绘，从头到脚完整呈现，双脚鞋子完整入镜，
      头顶保留适量留白空间，画面居中对称构图
画质与渲染：8k 分辨率，杰作，极致细节，专业角色设计表，
           清晰的材质纹理
```

### 模板字段说明

| 模块 | 字段 | 说明 |
|------|------|------|
| 基础特征 | 性别/年龄/体型/风格/色彩 | 定义角色整体定位 |
| 头部细节 | 头发/脸/眉毛/鼻子/眼神 | 注重面部特征与辨识度 |
| 身体穿搭 | 手/腿/衣/裤/鞋/配饰 | 衣物对称、剪裁清晰 |
| 技术环境 | 背景/姿势/比例/视角/光照/构图/画质 | 严格控制出图一致性 |

## 四、角色三视图

### 提示词

```
核心排版与视图：

一张极致干净的 3D 建模角色参考表 (Character Sheet)，
采用清晰的分屏排版布局。画面精确划分为左右两部分。

左侧区域（占画面 1/3）：
超大、超高清的头部与面部正式特写。头顶上方留有适当的
空白（负空间）。面部细节极致清晰，中性无表情，自然闭嘴。
完全露出颈部与下颌线轮廓。

右侧区域（占画面 2/3）：
包含三个并排的、标准的全身正交视图：正视图、正侧视图、
后视图。人物呈绝对标准的 A-Pose 站立，视角平视，
严禁任何透视畸变。

人物比例与特征：

采用 1:9 的超高修长头身比，下半身显著长于上半身
（腿与躯干比例为 5:4），极高腰线，修长纤细的四肢。
完全对称，清晰展现身体轮廓与衣物结构接缝。
```

### 排版布局示意

```
┌─────────────────────────────────┐
│          [角色三视图]             │
├──────────┬──────────────────────┤
│          │  正视图 │ 侧视图 │ 后视图 │
│  头部    │                      │
│  特写    │    A-Pose 全身正交视图  │
│  (1/3)   │                      │
│          │   1:9 修长头身比      │
├──────────┴──────────────────────┤
│ 比例：腿:躯干 = 5:4  ·  极高腰线 │
└─────────────────────────────────┘
```

## 五、场景图片生成

### 提示词

```
分析小说里所有的场景，参考以下模板，给出相应的生图提示词，
要求风格统一。

---
[核心设定]
风格：(例如：废土科幻、现代极简、中世纪哥特)
时间与天气：
色彩基调：

[空间与结构]
空间描述：
材质与细节：(例如：生锈的管道，光滑的水泥地面，破碎的玻璃)
远景/边界：(视野尽头或窗外的景象)

[光影与镜头]
光源设定：(光从哪里来?)
光影技术：(例如：丁达尔效应、全局光照、反射光)
视角：(例如：第一人称视角、无人机俯拍、广角仰视)
构图：(例如：绝对对称构图、三分法则、引导线构图)

[技术与约束]
画质与渲染：Unreal Engine 5，Octane Render，
           8k 分辨率，杰作，极高细节
附加要求：不要有角色出现在场景里
```

### 模板字段说明

| 模块 | 关键要素 |
|------|----------|
| 核心设定 | 风格基调、时间天气、色彩方向 |
| 空间结构 | 场景描述、材质细节、远景边界 |
| 光影镜头 | 光源位置、光影技术、视角构图 |
| 技术约束 | 渲染引擎、分辨率、无角色要求 |

## 整体工作流程

```
小说原文
    │
    ▼
① 小说内容改写  ─── 优化为推文文案，爆款开头 + 钩子结尾
    │
    ▼
② 分镜脚本      ─── 拆分为 10s/段的分镜，统一风格色调
    │
    ▼
③ 角色图片生成  ─── 按模板为每个角色生成生图提示词
    │
    ▼
④ 角色三视图    ─── 为每个角色生成标准三视图参考表
    │
    ▼
⑤ 场景图片生成  ─── 按模板为每个场景生成生图提示词
    │
    ▼
      最终产出：高质量 3D 精品漫剧
```
