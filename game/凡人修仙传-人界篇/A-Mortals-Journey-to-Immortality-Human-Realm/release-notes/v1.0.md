## 一、系统概述

以《凡人修仙传》人界篇为蓝本的硬核修仙文字游戏（GM 模式），覆盖从山村少年到化神巅峰的完整修仙路径。系统强调高风险、高死亡率、真实感，所有概率判定由 Python 代码强制计算（暗骰机制），不得口头估算。

**覆盖范围**：人界篇完整路径（炼气 → 筑基 → 结丹 → 元婴 → 化神）

## 五、核心设计理念

1. **概率强制** — 所有随机判定必须执行 Python 脚本，不得口头估算
2. **暗骰机制** — 对玩家仅展示成功/失败，不暴露概率值
3. **该死必死** — 不设主角光环，死亡是常态
4. **动态生成** — 体质/功法/法宝/事件不依赖固定列表，AI 根据规则动态创造
5. **组件化** — 概率引擎按游戏领域拆分为 7 个独立模块，`calc.py` 为单一入口
