## 二、新增函数

### `generate_world_intro(family_name, birthplace, special_event)`

**功能**：返回出身、出生地和特殊事件的类型标记，供 GM(AI) 动态生成世界介绍叙事。

**设计原则**：函数不包含任何固定数据（无预设姓名列表、无固定场地描述、无预设剧情钩子）。所有叙事内容——家族人物、环境描写、姓氏、关系、矛盾等——均由 AI 根据类型标记即时生成。

**参数**：

| 参数 | 类型 | 说明 |
|:----|:----|:------|
| `family_name` | str | 出身类型（如"修仙小族(旁支)"） |
| `birthplace` | str | 出生地类型（如"妖兽山脉边缘"） |
| `special_event` | str/tuple/list/dict | 特殊事件名称，支持多种传入格式 |

**返回值**：

```python
{
  "family_type": "修仙小族(旁支)",
  "birthplace_type": "妖兽山脉边缘",
  "special_event_name": "捡到残破玉简"
}
```

**支持的类型覆盖**：

- 出身类型：孤儿/弃婴、贫苦农家、猎户之家、渔/樵民家、商贾之家、书香门第、没落贵族、武林世家、江湖郎中/药师、修仙小族(旁支)、修仙大族(嫡系)、修仙大族(庶出)、被捡养/收养（共 13 种）
- 出生地：偏远山村、凡人城镇、修仙坊市附近、宗门属地村落、边境战区、妖兽山脉边缘、古战场遗址旁、灵气枯竭之地、灵气浓郁之地、绝境险地、流浪/无定所（共 11 种）
- 特殊事件：出生时天降异象、襁褓中放于宗门门口、幼年大病不死、被路过高人点拨、捡到残破玉简、家族被灭门、被当成炉鼎培养、出生在古墓/遗迹中、天生阴阳眼、被妖兽养大（共 10 种）

## 六、版本边界与限制

- `generate_world_intro` 仅输出类型标记，无属性加成，不影响游戏平衡性
- 所有叙事内容依赖 AI 生成能力，AI 输出的质量直接影响世界介绍体验
- 若 `family_name` 或 `birthplace` 无法识别，函数不会报错但返回空字符串标记
