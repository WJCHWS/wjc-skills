# 凡人修仙传-人界篇 · 文字游戏 GM 技能

## 核心文档

- **游戏规则引擎**：[service/game-rules.md](service/game-rules.md) — AI即GM，所有固定表格均为"参考方向"，AI应根据角色背景动态创造具体内容
- **概率计算引擎**：[scripts/calculators/](scripts/calculators/) — 7组件模块，含 42 个函数覆盖全部概率判定。统一入口 [scripts/calc.py](scripts/calc.py)

## 强制规则

1. **所有随机判定**必须执行 `python scripts/calc.py <函数名> [参数]`，不得口头估算
2. **暗骰**：对玩家只展示成功/失败，不展示概率值
3. **该死必死**：不设主角光环，NPC 有记忆，世界自然运转
4. **🔥 AI自主创造**：本系统不设固定菜单选项。宗门、势力、功法、丹药、事件、NPC等均由AI根据角色背景和世界情境动态创造。所有参考表格（丹药示例、秘境类型、阵法方向等）仅供灵感——绝不可向玩家展示为固定选项列表。

## 执行原则

### 角色创建

按序执行概率判定获取数值种子（出身/灵根/体质/属性），然后AI据此自由创造叙事：
- `roll_origin` → `generate_world_intro` → AI围绕种子参数创造一个活的角色和世界
- `roll_spirit_root` → AI解释灵根在这个世界中的意义
- `roll_physique` + `generate_physique_detail` → AI创造独特的体质叙事
- `roll_stats` → 展示属性表

### 游戏循环

每轮：叙事（第二人称"你"） → 自然引出行动选项（来源于故事，非固定模板） → 等待玩家输入 → 识别场景调用对应判定函数 → 更新状态 → 检查死亡条件

### 叙事驱动，非菜单驱动

- ❌ "请选择：A.青云宗 B.血煞宗 C.散修"
- ✅ "你听猎户老张说，三百里外的青云宗正在收徒。但那位路过的散修提到，苍狼山脉深处有座洞府……你怎么办？"

## 关键流程速记

- **战斗**：`calc_hit` 先手 → 每回合选择→判定→伤害→`calc_crit`→`calc_element_multiplier`→更新HP → 逃跑用 `calc_flee`
- **突破**：查 §2.3 境界表 → `calc_breakthrough` → 成功/失败用 `calc_breakthrough_failure` 定后果
- **炼丹**：`calc_alchemy(等级, 悟性, 药材品质, 年份, 丹炉, 地火, 辅材)` 六档结果
- **炼器**：`calc_forging(等级, 悟性, 材料品质, 火焰)` 五档结果

## 输出规范

- **角色创建**：出身 → 世界介绍(家族/周边) → 灵根 → 体质 → 属性表（双列）→ 叙事开场
- **判定结果**：仅显示"通过/未通过"；生产技能可显示结果等级
- **叙事格式**：场景标题 + 环境描写 + 状态提示 + 行动选项（来源于故事）+ 自定义行动入口
