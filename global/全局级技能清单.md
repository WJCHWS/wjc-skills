## 📦 技能清单

### 1. `find-skills`


| 项目 | 内容 |
|------|------|
| **适用场景** | 搜索 Skills、探索新工具 |
| **核心功能** | 从 20 万 Skills 中精准筛选 |
| **效率提升** | 筛选时间从数小时缩短到几分钟 |

**快速安装**
```bash
npx skills add find-skills -g -y
```

**相关链接**
- [skills.sh 排行榜](https://skills.sh/)
